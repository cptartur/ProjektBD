# Projekt bazy danych MPK

Projekt bazy danych obsługującej miejskie przedsiębiorstwo komunikacyjne, przygotowany na potrzeby przedmiotu Bazy danych.

## Technologie

Projekt w całości napisany w języku T-SQL
